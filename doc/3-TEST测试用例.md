# MyTodos 团队待办 App — 测试用例文档

## 1. 文档概述

### 1.1 文档目的

本文档基于《1-SRS需求.md》和《2-PRD设计.md》，为 MyTodos 定义测试用例，覆盖所有功能需求（FR）的验证场景，包括正常路径、异常路径和权限边界。

### 1.2 测试范围

- 功能测试：覆盖 FR-001 ~ FR-006, FR-101 ~ FR-106
- 权限测试：覆盖角色权限矩阵
- 接口测试：Gitee API 交互与错误处理
- 并发测试：多端冲突场景
- UI 测试：页面跳转与组件交互

### 1.3 测试环境

- 测试设备：Android 中端机 / iOS 中端机 / Windows 桌面端
- 网络环境：正常网络 / 弱网(模拟) / 断网
- 测试数据：预置标准工作区（含 4 成员、11 标签、若干任务）
